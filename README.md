# scoop-bucket

Scoop bucket for richard-fairthorne's projects.

## Installation

```powershell
scoop bucket add richard-fairthorne https://github.com/richard-fairthorne/scoop-bucket
scoop install pairqr
```

## Available Apps

| App | Description |
|-----|-------------|
| [pairqr](https://github.com/richard-fairthorne/pairqr) | Pair Android devices for wireless ADB debugging via QR code |
